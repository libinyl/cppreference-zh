对 C 库的扩展部分 II ：动态分配函数 (Extensions to the C Library Part II: Dynamic Allocation Functions) ， ISO/IEC TR 24731-2:2010 ，为 C 标准库定义下列新组件：

__STDC_ALLOC_LIB__
 指示一致性等级的 long 类型常量
(宏常量)
定义于头文件 <stdio.h>
fmemopen
(动态内存 TR)
 把固定大小的内存缓冲区作为 I/O 流打开
(函数)
open_memstreamopen_wmemstream
(动态内存 TR)
 把动态改变大小的内存缓冲区作为 I/O 流打开
(函数)
asprintfaswprintfvasprintfvaswprintf
(动态内存 TR)
 sprintf 等的变体，写入自动分配的缓冲区并返回指向它的指针
(函数)
getlinegetwlinegetdelimgetwdelim
(动态内存 TR)
 从流读入至动态改变大小的缓冲区，直到分隔符/行尾
(函数)
定义于头文件 <string.h>
strdup
(动态内存 TR)
 分配字符串的副本
(函数)
strndup
(动态内存 TR)
 分配字符串副本，至多到指定的大小
(函数)

此库扩展亦引入赋值分配字符 m ，用于 fscanf 和 fwscanf 函数族中的 %s 、 %[ 及 %c 转换指定符。

注意

函数 fmemopen 、 open_memstream 、 open_wmemstream 、 getdelim 、 getline 、 strdup 、 strndup 及对 fscanf 的扩展可用于 POSIX (ISO/IEC 9945:2003) 。

函数 asprintf 及 vasprintf 可用于 Linux 标准基础 (Linux Standard Base) (ISO/IEC IS 23360:2006) 。
