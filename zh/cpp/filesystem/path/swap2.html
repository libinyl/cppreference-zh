void swap( path& lhs, path& rhs ) noexcept;
 (C++17 起)

交换 lhs 与 rhs 的状态。等效于调用 lhs.swap(rhs) 。

参数

lhs, rhs - 要交换状态的 path

返回值

（无）

参阅
