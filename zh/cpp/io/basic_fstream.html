template<

    class CharT,
    class Traits = std::char_traits<CharT>

> class basic_fstream : public std::basic_iostream<CharT, Traits>

类模板 basic_fstream 实现基于文件的流上的高层输入/输出。它将 std::basic_iostream 的高层接口赋予基于文件的缓冲（ std::basic_filebuf ）。

std::basic_fstream 的典型实现只保有一个非导出数据成员： std::basic_filebuf<CharT, Traits> 的实例。

示例

运行此代码
#include <iostream>
#include <fstream>
#include <string>
 
int main() {
  std::string filename = "test.bin";
  std::fstream s(filename, s.binary | s.trunc | s.in | s.out);
  if (!s.is_open()) {
    std::cout << "failed to open " << filename << '\n';
  } else {
    // 写入
    double d = 3.14;
    s.write(reinterpret_cast<char*>(&d), sizeof d); // 二进制输出
    s << 123 << "abc";                              // 文本输出
 
    // 对于 fstream ，这会移动文件位置指针（放置与获取）
    s.seekp(0);
 
    // 读取
    s.read(reinterpret_cast<char*>(&d), sizeof d); // 二进制输入
    int n;
    std::string str;
    if (s >> n >> str)                             // 文本输入
      std::cout << "read back from file: " << d << ' ' << n << ' ' << str << '\n';
  }
}

输出： 

read back from file: 3.14 123 abc
