定义于头文件 <ios>
class failure;

类 std::ios_base::failure 定义输入/输出库中的函数在失败时抛出的异常对象。

std::ios_base::failure 可以定义为 std::ios_base 的成员类，或拥有等价功能的另一个类的同义词（ typedef ）。 (C++17 起)

示例

运行此代码
#include <iostream>
#include <fstream>
int main()
{
    std::ifstream f("doesn't exist");
    try {
        f.exceptions(f.failbit);
    } catch (const std::ios_base::failure& e)
    {
        std::cout << "Caught an ios_base::failure.\n"
                  << "Explanatory string: " << e.what() << '\n'
                  << "Error code: " << e.code() << '\n';
    }
}

输出： 

Caught an ios_base::failure.
Explanatory string: ios_base::clear: unspecified iostream_category error
Error code: iostream:1
