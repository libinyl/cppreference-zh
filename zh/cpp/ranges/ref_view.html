定义于头文件 <ranges>
template<Range R>

    requires std::is_object_v<R>

class ref_view : public ranges::view_interface<ref_view<R>>
 (C++20 起)

ref_­view 是某个其他 range 的元素的 view 。它包装到该 Range 的引用。

数据成员

std::ranges::ref_view::r_

R* r_ = nullptr; /* exposition-only */

指向被引用范围的指针

成员函数

std::ranges::ref_view::ref_view

constexpr ref_view() noexcept = default;
 (1)
template<__NotSameAs<ref_view> T>

    requires /* see below */

constexpr ref_view(T&& t);
 (2)
1) 以 nullptr 初始化 r_ 。默认初始化的 ref_view 不引用 range 。
2) 以 std::addressof(static_­cast<R&>(std::forward<T>(t))) 初始化 r_ 。

__NotSameAs 是仅为说明的辅助概念。

(2) 的 requires 子句中的表达式等价于 ConvertibleTo<T, R&> && requires { FUN(declval<T>()); } ，其中仅为说明的函数 FUN 声明为 void FUN(R&); void FUN(R&&) = delete; 。

t - 要引用的范围

非成员函数

begin, end(std::ranges::ref_view)

friend constexpr iterator_t<R> begin(ref_view r);
 (1)
friend constexpr sentinel_t<R> end(ref_view r);
 (2)
1) 等价于 return r.begin();
2) 等价于 return r.end();

这些函数对通常无限定或有限定查找不可见，而且只能在 std::ranges::ref_view<R> 为参数的关联类时由参数依赖查找找到。
