进行 optional 对象上的比较。

1-6) 比较二个 optional 对象， lhs 和 rhs 。仅若 lhs 与 rhs 都含值，才比较容纳的值（使用 T 对应的运算符）。否则，
  • 若且唯若 lhs 与 rhs 都不含值，才认为 lhs 等于 rhs 。
  • 若且唯若 rhs 含值且 lhs 不含值，才认为 lhs 小于 rhs 。
7-18) 比较 opt 与 nullopt 比较。与 (1-6) 中与不含值的 optional 比较时等价。
19-30) 比较 opt 与value 。仅若 opt 含值才比较值。否则认为 opt 小于 value 。若 *opt 与 value 间的对应比较表达式非良式，或若结果不能转换为 bool ，则行为未定义。

参数

lhs, rhs, opt - 要比较的 optional 对象
value - 与所含值比较的值

返回值

1) 若 bool(lhs) != bool(rhs) ，返回 false ；

否则，若 bool(lhs) == false （且 bool(rhs) == false 亦然），返回 true ；

否则，返回 *lhs == *rhs。
2) 若 bool(lhs) != bool(rhs) ，返回 true ；

否则，若 bool(lhs) == false （且 bool(rhs) == false 亦然），返回 false ；

否则，返回 *lhs != *rhs 。
3) 若 bool(rhs) == false ，返回 false ；

否则，若 bool(lhs) == false ，返回 true ；

否则，返回 *lhs < *rhs 。
4) 若 bool(lhs) == false ，返回 true ；

否则，若 bool(rhs) == false ，返回 false ；

否则，返回 *lhs <= *rhs 。
5) 若 bool(lhs) == false ，返回 false ；

否则，若 bool(rhs) == false ，返回 true ；

否则，返回 *lhs > *rhs 。
6) 若 bool(rhs) == false ，返回 true ；

否则，若 bool(lhs) == false ，返回 false ；

否则，返回 *lhs >= *rhs 。
7-8) 返回 !opt 。
9-10) 返回 bool(opt) 。
11) 返回 false 。
12) 返回 bool(opt) 。
13) 返回 !opt 。
14) 返回 true 。
15) 返回 bool(opt) 。
16) 返回 false 。
17) 返回 true 。
18) 返回 !opt 。
19) 返回 bool(opt) ? *opt == value : false 。
20) 返回 bool(opt) ? value == *opt : false 。
21) 返回 bool(opt) ? *opt != value : true 。
22) 返回 bool(opt) ? value != *opt : true 。
23) 返回 bool(opt) ? *opt < value  : true 。
24) 返回 bool(opt) ? value < *opt  : false 。
25) 返回 bool(opt) ? *opt <= value : true 。
26) 返回 bool(opt) ? value <= *opt : false 。
27) 返回 bool(opt) ? *opt > value  : false 。
28) 返回 bool(opt) ? value > *opt  : true 。
29) 返回 bool(opt) ? *opt >= value : false 。
30) 返回 bool(opt) ? value >= *opt : true 。

异常

1-6) （无）
19-30) （无）
