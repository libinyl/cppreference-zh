constexpr basic_string_view
    substr(size_type pos = 0, size_type count = npos ) const;
 (C++17 起)

返回子串 [pos, pos + rcount) 的视图，其中 rcount 是 count 与 size() - pos 中较小者。

参数

pos - 首字符的位置
count - 请求的长度

返回值

子串 [pos, pos + rcount) 的视图。

异常

若 pos > size() 则抛出 std::out_of_range 。

复杂度

常数。

参阅

copy
 复制字符
(公开成员函数)
find
 在视图中查找字符
(公开成员函数)
