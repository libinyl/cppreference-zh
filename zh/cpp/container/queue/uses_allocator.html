template< class T, class Container, class Alloc >

struct uses_allocator<queue<T,Container>,Alloc> :

    std::uses_allocator<Container, Alloc>::type { };
 (C++11 起)

为 std::queue 提供 std::uses_allocator 类型特性的通透特化：容器适配器使用分配器，若且唯若底层容器使用。
