local_iterator end( size_type n );
 (C++11 起)
const_local_iterator end( size_type n ) const;
 (C++11 起)
const_local_iterator cend( size_type n ) const;
 (C++11 起)

返回后随下标为 n 的桶的最后元素的元素的迭代器。此元素表现为占位符，试图访问它会导致未定义行为。

参数

n - 要访问的桶的下标

返回值

指向后随最后元素的元素的迭代器。

复杂度

常数

参阅

begin(size_type) cbegin(size_type)
 返回一个迭代器，指向指定的桶的开始
(公开成员函数)
