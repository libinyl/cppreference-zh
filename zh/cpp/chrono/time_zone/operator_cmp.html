以名称比较二个 time_zone 对象 x 与 y 。

返回值

1) x.name() == y.name() 。
2) x.name() != y.name() 。
3) x.name() < y.name() 。
4) x.name() > y.name() 。
5) x.name() <= y.name() 。
6) x.name() >= y.name() 。
