from_stream
 
template <class CharT, class Traits, class Alloc = std::allocator<CharT>>

std::basic_istream<CharT, Traits>&
from_stream(std::basic_istream<CharT, Traits>& is, const CharT* fmt,
            std::chrono::year_month& ym, std::basic_string<CharT, Traits, Alloc>* abbrev = nullptr,

            std::chrono::minutes* offset = nullptr);
 (C++20 起)

试图按照格式字符串 fmt 分析输入流 is 到 std::chrono::year_month ym 中。

表现为无格式输入函数 (UnformattedInputFunction) ，除了它在 is.gcount() 上拥有未指定的效果。在构造并检查 sentry 对象后，试图按照格式字符串 fmt 分析输入流 is 到 std::chrono::year_month ym 中。若分析无法解码出合法的 year_month ，则调用 is.setstate(std::ios_base::failbit) 并且不修改 ym 。

若使用并成功分析了格式指定符 %Z ，则若 abbrev 非空则赋分析的值给 *abbrev 。若使用并成功分析了格式指定符 %z 或修改的变体，则若 offset 非空则赋分析的值给 *offset 。

格式字符串

格式字符串由零或更多个转换指定符和通常字符组成。每个通常字符，除了空白字符和空终止字符，都匹配一个来自输入流的等同字符，或若流上的下个字符不比较相等则导致函数失败。

每个空白字符匹配输入流中的零或更多个空白字符。

每个不修饰的转换指定符以 % 字符后随一个确定指定符行为的字符开始。某些转换指定符拥有在 % 字符后插入修饰字符 E 或 O 的修饰形式。某些转换指定符拥有在 % 字符后插入作为十进制整数（下面显示为 N ）的宽度参数的修饰形式。每个转换指定符导致匹配的字符被按照下表转译成时间和日期类型的部分。

格式字符串中，以 % 开始但不匹配下列转换指定符之一的字符序列被转译成通常字符。

若 from_stream 无法分析格式字符串所指定的每项内容，或若分析出的信息不足以指定完整结果，或若分析透露矛盾的信息，则调用 is.setstate(std::ios_base::failbit) 。
