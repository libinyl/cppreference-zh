语句（statement）是依序执行的 C++ 程序片段。任何函数体都是语句的序列。例如： 

int main()
{
    int n = 1;                        // 声明语句
    n = n + 1;                        // 表达式语句
    std::cout << "n = " << n << '\n'; // 表达式语句
    return 0;                         // return 语句
}

C++ 包含下列类型的语句：

1) 表达式语句（expression statement）；
2) 复合语句（compound statement）；
3) 选择语句（selection statement）；
4) 循环语句（iteration statement）；
5) 跳转语句（jump statement）；
6) 声明语句（declaration statement）；
7) try 块；
8) atomic 与 synchronized 块(TM TS)。

标号

通过在语句自身之前提供后随冒号的标号，任何语句都可带有标号。

attr(可选) 标识符 : 语句 (1)
attr(可选) case 常量表达式 : 语句 (2)
attr(可选) default : 语句 (3)
1) goto 的目标
2) switch 语句中的 case 标号;
3) switch 语句中的 default 标号

属性序列 attr 可以紧接标号之前出现（这种情况下它应用到标号），或紧跟在语句自身之后出现，这种情况下它应用到整条语句。一条语句可以带有多个标号。标号（且只有标号）拥有函数作用域。无限定查找中忽略标号：标号可以与程序中的任何其他实体具有相同名字。

表达式语句

表达式后跟一个分号是一条语句。

attr(可选) 表达式(可选) ; (1)
attr(C++11) - 可选的任意数量属性的序列
表达式 - 一个表达式

典型 C++ 程序的大部分语句都是表达式语句，例如赋值或函数调用。

无表达式的表达式语句被称作空语句（null statement）。它通常用来为 for 或 while 循环提供空体。它亦可用于在复合语句尾部携带标号。

复合语句

复合语句或（代码）块是花括号环绕的语句序列。

attr(可选) { 语句...(可选) } (1)

当期待一条语句，但需要依序执行多条语句时（例如在 if 语句或循环中）可使用复合语句： 

if (x > 5)          // if 语句的开始
{                   // 块的开始
    int n = 1;      // 声明语句
    std::cout << n; // 表达式语句
}                   // 块的结束，if 语句的结束

每个复合语句都引入其自身的块作用域；声明于块中的变量在闭花括号处以逆序销毁： 

int main()
{
    {                                // 块的开始
        std::ofstream f("test.txt"); // 声明语句
        f << "abc\n";                // 表达式语句
    }                                // 块结束：冲洗并关闭 f
    std::ifstream f("test.txt"); 
    std::string str;
    f >> str; 
}

选择语句

选择语句在数个控制流中选择一个。

attr(可选) if ( 条件 ) 语句 (1)
attr(可选) if ( 条件 ) 语句 else 语句 (2)
attr(可选) switch ( 条件 ) 语句 (3)
(C++17 前)
attr(可选) if constexpr(可选) ( 初始化语句(可选) 条件 ) 语句 (1)
attr(可选) if constexpr(可选) ( 初始化语句(可选) 条件 ) 语句 else 语句 (2)
attr(可选) switch ( 初始化语句(可选) 条件 ) 语句 (3)
(C++17 起)
1) if 语句；
2) 带 else 子句的 if 语句；
3) switch 语句。

循环语句

循环语句重复执行一些代码。

attr(可选) while ( 条件 ) 语句 (1)
attr(可选) do 语句 while ( 表达式 ) ; (2)
attr(可选) for ( 初始化语句 条件(可选) ; 表达式(可选) ) 语句 (3)
attr(可选) for ( for-范围声明 : for-范围初始化器 ) 语句 (4) (C++11 起)
1) while 循环；
2) do-while 循环；
3) for 循环；
4) 范围 for 循环。

跳转语句

跳转语句无条件地转移控制流

attr(可选) break ; (1)
attr(可选) continue ; (2)
attr(可选) return 表达式(可选) ; (3)
attr(可选) return 花括号初始化器列表 ; (4) (C++11 起)
attr(可选) goto 标识符 ; (5)
1) break 语句；
2) continue 语句；
3) 带可选表达式的 return 语句；
4) 使用列表初始化的 return 语句；
5) goto 语句。

注意：对于所有跳转语句，转移出循环，出块，或回到被初始化的具有自动存储期的变量之前，会牵涉到对在转移发起点处于作用域中，但在转移目标点不在作用域的，具有自动存储期的对象的销毁。若初始化了多个对象，则销毁顺序与初始化顺序相反。

声明语句

声明语句在块中引入一个或多个标识符。

块声明 (1)
1) 细节见声明及初始化。

try 块

try 块提供当执行其他语句时捕获其所抛出的异常的能力。

attr(可选) try 复合语句 处理块序列 (1)
1) 细节见 try/catch。

atomic 与 synchronized 块

atomic 与 synchronized 块用来实现事务性内存。

synchronized 复合语句 (1) (TM TS)
atomic_noexcept 复合语句 (2) (TM TS)
atomic_cancel 复合语句 (3) (TM TS)
atomic_commit 复合语句 (4) (TM TS)
1) synchronized 块，与所有 synchronized 块在一个全序中执行；
2) 在发生异常时中止的 atomic 块；
3) 在发生异常时回滚的 atomic 块；
4) 在发生异常时提交的 atomic 块。
 (TM TS)
