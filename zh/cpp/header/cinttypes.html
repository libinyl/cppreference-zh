此头文件原作为 <inttypes.h> 存在于 C 标准库。

提供从 C 字节字符串与宽字符串到 std::intmax_t 与 std::uintmax_t 的转换，为 std::intmax_t 重载一些数学函数，并为声明于 <cstdint> 的各个类型提供 C 风格输入/输出的格式宏。
