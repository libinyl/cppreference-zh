此头文件原作为 <stdio.h> 存在于 C 标准库。

此头文件是 C 风格输入/输出库的一部分。

类型

FILE 类型，足以保有控制 C I/O 流所需的全部信息
fpos_t 类型，足以唯一指定文件中的位置
size_t
 sizeof 运算符返回的无符号整数类型
(typedef)

宏

注意

  • NULL 亦定义于下列头文件：
    • <clocale>
    • <ctime>
    • <cstddef>
    • <cstring>
    • <cwchar>
    • <cstdlib>
  • std::size_t 亦定义于下列头文件：
    • <ctime>
    • <cstddef>
    • <cstring>
    • <cwchar>
    • <cstdlib>
