此头文件原作为 <errno.h> 存在于 C 标准库。

此头文件是错误处理库的一部分。

注意

头文件 <cerrno> 基于 C 标准库头文件 errno.h ，然而 C++ 主要从 POSIX 标准，而非 C 标准库采用 <cerrno> 所定义的宏。
