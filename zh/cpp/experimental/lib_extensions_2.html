C++ 库基础扩展的版本 2 (Version 2 of the C++ Extensions for Library Fundamentals) ， ISO/IEC TS 19568:2017 在定义于版本 1 中的组件外，还为 C++ 标准库定义下列组件：


C++ 检测手法

定义于头文件 <experimental/type_traits>
is_detected detected_t detected_or
 检测 template-id 是否良式的别名模板
(别名模板)
nonesuch
 detected_t 在失败时返回的类类型
(类)

传播 const 的仿指针对象包装器

定义于头文件 <experimental/propagate_const>
propagate_const
 传播 const 的仿类指针对象包装器
(类模板)

非占有指针

定义于头文件 <experimental/memory>
observer_ptr
 表示非占有指针的类
(类模板)

std::array 创建

定义于头文件 <experimental/array>
make_array
 创建 std::array 对象，从参数推导出其大小和可选的元素类型
(函数模板)

ostream_joiner

定义于头文件 <experimental/iterator>
ostream_joiner
 写入相继元素到输出流，并以分隔符分隔相邻元素的输出迭代器
(类模板)

简单的随机数生成设施

为每个线程维护一个 std::default_random_engine 类型的逐线程引擎，初始化到不可预测状态，而它并为下列函数所用。

定义于头文件 <experimental/random>
randint
 生成指定范围中的随机整数
(函数模板)
reseed
 重播种单个线程的随机引擎
(函数)
定义于头文件 <experimental/algorithm>
sample
 从序列随机选择 n 个元素
(函数模板)
shuffle
 随机地重排范围中的元素
(函数模板)

特性测试宏

定义于头文件 <experimental/type_traits>
__cpp_lib_experimental_logical_traits
 至少为 201511 的值，指示支持逻辑运算符类型特性
(宏常量)
__cpp_lib_experimental_detect
 至少为 201505 的值，指示支持检测手法
(宏常量)
定义于头文件 <experimental/propagate_const>
__cpp_lib_experimental_propagate_const
 至少为 201505 的值，指示支持 propagate_const
(宏常量)
定义于头文件 <experimental/functional>
__cpp_lib_experimental_not_fn
 至少为 201406 的值，指示支持 not_fn
(宏常量)
定义于头文件 <experimental/memory>
__cpp_lib_experimental_observer_ptr
 至少为 201411 的值，指示支持 observer_ptr
(宏常量)
定义于头文件 <experimental/array>
__cpp_lib_experimental_make_array
 至少为 201505 的值，指示支持 make_array
(宏常量)
定义于头文件 <experimental/iterator>
__cpp_lib_experimental_ostream_joiner
 至少为 201411 的值，指示支持 ostream_joiner
(宏常量)
定义于头文件 <experimental/vector>
__cpp_lib_experimental_erase_if
 至少为 201411 的值，指示支持统一容器擦除
(宏常量)
定义于头文件 <experimental/numeric>
__cpp_lib_experimental_gcd_lcm
 至少为 201411 的值，指示支持 gcd 和 lcm
(宏常量)
定义于头文件 <experimental/random>
__cpp_lib_experimental_randint
 至少为 201511 的值，指示支持 randint
(宏常量)
定义于头文件 <experimental/source_location>
__cpp_lib_experimental_source_location
 至少为 201505 的值，指示支持 source_location
(宏常量)

并入 C++17

库基础 v2 的下列组件已被采纳到 C++17 标准。

定义于头文件 <experimental/type_traits>
void_t
 void 的变长别名模板
(别名模板)
定义于头文件 <experimental/type_traits>
conjunction
 变长逻辑与元函数
(类模板)
disjunction
 变长逻辑或元函数
(类模板)
negation
 逻辑非元函数
(类模板)
定义于头文件 <experimental/functional>
not_fn
 创建函数对象，它返回其所保有的函数对象结果的逻辑补
(函数模板)
定义于头文件 <experimental/numeric>
gcd
 返回二个整数最大公约数的 constexpr 函数模板
(函数模板)
lcm
 返回二个整数最小公倍数的 constexpr 函数模板
(函数模板)
