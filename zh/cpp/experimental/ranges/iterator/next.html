前进迭代器 i n 次，或直至抵达 bound ，先到为止，并返回前进的迭代器。

1) 等价于 ++i; return i; 。
2) 等价于 ranges::advance(i, n); return i; 。
3) 等价于 ranges::advance(i, bound); return i; 。
4) 等价于 ranges::advance(i, n, bound); return i; 。

定制点对象

名称 ranges::next 代表一个定制点对象，它是字面 Semiregular 类类型的函数对象（为说明目的以 NextT 表示）。NextT 的所有实例均相等。从而能自由地复制 ranges::next ，且能交替使用其副本。

给定类型集合 Args... ，若 std::declval<Args>()... 满足上面给 ranges::next 的参数要求，则 NextT 将满足 ranges::Invocable<const NextT, Args...> 。否则，没有 NextT 的函数调用运算符参与重载决议。

在每个定义 ranges::next 的翻译单元中，它都指代定制点对象的同一实例。（这表示能自由地在类似 inline 函数和函数模板的东西中使用它，而不违反单一定义规则。）

返回值

前进的迭代器。

注意

虽然对重载 (2) 忽略 n 表现为如同 n 为 1 ，对重载 (4) 忽略 n 表现为 n 为无穷大（始终前进到 bound ）。
