template< class W >
void swap( observer_ptr<W>& lhs, observer_ptr<W>& rhs ) noexcept;
 (库基础 TS v2)

为 std::experimental::observer_ptr 特化 swap 算法。交换 lhs 与 rhs 的指针。调用 lhs.swap(rhs) 。

参数

lhs, rhs - 要交换内容的 observer_ptr

返回值

（无）

复杂度

常数

参阅

swap
 交换两个对象的值
(函数模板)
swap
 交换所查看的对象
(公开成员函数)
