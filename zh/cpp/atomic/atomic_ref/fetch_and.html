T fetch_and( T arg,
             std::memory_order order = std::memory_order_seq_cst ) const noexcept;
 (C++20 起)
(仅为 atomic_ref<Integral> 模板特化的成员)

原子地以值和 arg 的逐位与结果替换被引用对象的当前值。此操作是读修改写操作。按照 order 的值影响内存。

参数

arg - 逐位与的另一参数
order - 要施加的内存制约

返回值

被引用对象立即于此函数效果前的值。
